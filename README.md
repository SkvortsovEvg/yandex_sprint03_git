* `untracked` (–∞–Ω–≥–ª. ¬´–Ω–µ–æ—Ç—Å–ª–µ–∂–∏–≤–∞–µ–º—ã–π¬ª)

–í–æ—Ç —á—Ç–æ –µ—â—ë –≤–∞–∂–Ω–æ —É—á–µ—Å—Ç—å:

–ë–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —Ñ–∞–π–ª–æ–≤ –≤ —Ç–∏–ø–∏—á–Ω–æ–º –ø—Ä–æ–µ–∫—Ç–µ –±—É–¥—É—Ç –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–∏ `tracked` (—Ç–æ –µ—Å—Ç—å –∑–∞–∫–æ–º–º–∏—á–µ–Ω—ã –∏ –Ω–µ –∏–∑–º–µ–Ω–µ–Ω—ã –ø–æ—Å–ª–µ –∫–æ–º–º–∏—Ç–∞). –í—ã –Ω–µ —É–≤–∏–¥–∏—Ç–µ —ç—Ç–æ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –≤ –≤—ã–≤–æ–¥–µ –∫–æ–º–∞–Ω–¥—ã `git status` ‚Äî –∏–Ω–∞—á–µ –æ–Ω–∞ –±—ã –∫–∞–∂–¥—ã–π —Ä–∞–∑ –≤—ã–≤–æ–¥–∏–ª–∞ —Å–ø–∏—Å–æ–∫ –≤–æ–æ–±—â–µ –≤—Å–µ—Ö —Ñ–∞–π–ª–æ–≤ –ø—Ä–æ–µ–∫—Ç–∞. –í –∏—Ç–æ–≥–µ `git status` –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Ç–æ–ª—å–∫–æ —Å–ª–µ–¥—É—é—â–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è —Ñ–∞–π–ª–æ–≤:

–î–∞–ª—å—à–µ –≤—ã –±—É–¥–µ—Ç–µ –¥–æ–±–∞–≤–ª—è—Ç—å –≤ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π —Ñ–∞–π–ª—ã –∏ –ø—Ä–æ–≤–µ—Ä—è—Ç—å –∏—Ö —Å—Ç–∞—Ç—É—Å—ã. –ú—ã —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º —á–µ—Ç—ã—Ä–µ –ø—Ä–∏–º–µ—Ä–∞ —Å–æ—Å—Ç–æ—è–Ω–∏–π, –≤ –∫–æ—Ç–æ—Ä—ã—Ö –º–æ–∂–µ—Ç –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤–∞—à —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π.

1. **–ù–µ—Ç –Ω–∏ `staged`-, –Ω–∏ `modified`-, –Ω–∏ `untracked`-—Ñ–∞–π–ª–æ–≤.**
